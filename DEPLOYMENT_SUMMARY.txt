## SUPPORT & TROUBLESHOOTING

**Common Issues**:

1. **Window doesn't appear**
   - Solution: Check TEST_RESULTS.txt (should show PASS)
   - Try: Running in Dev Mode for console output
   - Verify: GPU flag is enabled (check main.js line 6)

2. **Installation fails**
   - Solution: Run Install.bat as Administrator
   - Ensure: Write permissions to Program Files

3. **Can't connect to peer**
   - Verify: Correct IP address or hostname
   - Check: Both systems on same network
   - Review: Firewall settings if needed

4. **Application crashes**
   - Check: Console output in Dev Mode
   - Review: System requirements
   - Verify: Node.js/npm installed (for non-exe versions)

**For Help**:
1. Read: INSTALL_GUIDE.md
2. Check: TEST_RESULTS.txt
3. Review: Console output (Dev Mode)
4. Read: All documentation files
